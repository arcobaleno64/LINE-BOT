## 1. 歡迎使用

### 1.1 關於本 Bot

您好！我是您的智能助手，專為協助您處理日常資訊需求而設計。

**我能幫您：**
- 回答各種問題
- 分析圖片內容
- 處理文件並回答相關問題
- 提供即時時間資訊
- 查詢最新資訊

**我的特色：**
- 說話親切有禮
- 回答簡潔實用
- 使用繁體中文

## 2. 快速開始

### 2.1 如何開始對話

1. 在 LINE 中找到本 Bot
2. 點擊「開始」或直接發送訊息
3. 等待回覆後即可開始對話

### 2.2 基本操作

| 操作 | 方法 |
|------|------|
| 提問 | 直接輸入問題，無需加上特殊指令 |
| 傳送圖片 | 直接發送圖片（可附加問題） |
| 傳送檔案 | 直接發送支援格式的檔案 |
| 中斷對話 | 等待一段時間不說話，系統會自動重置對話 |

### 2.3 首次使用建議

初次使用，建議您可以試試：
- 問一個簡單問題：「今天天氣如何？」
- 查詢時間：「現在幾點？」
- 傳送一張圖片問問題

## 3. 功能詳解

### 3.1 文字對話

**使用方式：** 直接輸入問題或指令

**範例：**
| 您輸入 | Bot 回覆 |
|--------|----------|
| 請幫我解釋什麼是人工智慧 | 會用簡單易懂的方式說明 |
| 推薦我一部好看的電影 | 根據您的需求推薦 |
| 如何煮咖啡？ | 提供步驟說明 |

**小技巧：** 問題越清楚，回覆越準確

---

### 3.2 圖片分析

**使用方式：** 傳送圖片 + 說明您想知道的事

**範例：**
| 您輸入 | Bot 回覆 |
|--------|----------|
| 傳送圖片 + 「這張圖裡有什麼？」 | 描述圖片內容 |
| 傳送截圖 + 「幫我翻譯這段文字」 | 提供翻譯 |
| 傳送產品圖片 + 「這個多少錢？」 | 說明無法查詢價格，建議上官網 |

**限制：**
- 僅限一對一聊天
- 群組/群聊中無法使用此功能

---

### 3.3 檔案處理

**支援的檔案格式：**
| 格式 | 副檔名 | 說明 |
|------|--------|------|
| 文字檔 | .txt | 純文字文件 |
| 網頁檔 | .md | Markdown 格式 |
| 資料表 | .csv | 逗號分隔資料 |
| 設定檔 | .json, .xml | 結構化資料 |
| 日誌檔 | .log | 系統日誌 |
| PDF | .pdf | 文件檔案 |

**使用方式：**
1. 傳送檔案
2. 說明您想做的事

**範例：**
| 您輸入 | Bot 回覆 |
|--------|----------|
| 傳送 .txt + 「摘要這份文件」 | 提供文件摘要 |
| 傳送 .pdf + 「這份報告的重點是什麼？」 | 分析並回答 |
| 傳送 .csv + 「幫我統計資料」 | 說明統計結果 |

**關於 PDF：**
- 僅支援**文字型 PDF**（可直接複製文字的 PDF）
- 掃描圖片型 PDF 暫時無法處理

**檔案下載：**
- 處理完成後，您可能會收到一個**下載連結**
- 連結有效期為 **24 小時**
- 請在期限內下載檔案

**限制：**
- 僅限一對一聊天
- 群組/群聊中無法使用此功能

---

### 3.4 時間查詢（快捷指令）

**使用方式：** 直接輸入時間相關指令

| 您輸入 | Bot 回覆 |
|--------|----------|
| 現在幾點 | 現在時間：HH:mm:ss |
| 今天幾號 | 今天日期：yyyy 年 MM 月 dd 日 |
| 今天星期幾 | 今天：星期X |
| 明天星期幾 | 明天：星期X |

**特點：** 查詢時間**不會使用 AI**，回覆快速準確

---

### 3.5 最新資訊查詢

**使用方式：** 輸入問題時加入時間關鍵字

| 關鍵字範例 | 適用問題 |
|-----------|----------|
| 最新、即時 | 「最新 AI 新聞」、「即時天氣」 |
| 今天、本日 | 「今天新聞」、「本日匯率」 |
| 最近、最新消息 | 「最新資訊」、「最新公告」 |

**範例：**
| 您輸入 | Bot 回覆 |
|--------|----------|
| 幫我查最新科技新聞 | 搜尋並整理最新資訊 |
| 最新天氣如何？ | 提供最新天氣資訊 |

**注意：** 此功能需要網路搜尋服務，若未啟用則回覆將以一般對話處理

---

### 3.6 群組/群聊使用

**使用方式：** 在群組中需要**@提及 Bot** 才會回覆

| 情境 | 是否回覆 |
|------|----------|
| 一對一聊天 | 直接回覆 |
| 群組被 @mention | 回覆 |
| 群組沒有 @mention | 不回覆 |
| 群組傳送圖片/檔案 | 不處理 |

**範例：**
```
@Bot名稱 今天午餐推薦什麼？
-> Bot 會回覆推薦
```

## 4. 限制與注意事項

### 4.1 訊息頻率限制

為確保服務穩定，同一時間內傳送過多訊息可能會被暫時限制。

**當觸發限制時：** 您會收到「訊息有點密集，請稍後再試」

**解決方式：** 等待幾秒後再繼續使用

### 4.2 AI 服務限制

當 AI 服務忙碌時，可能會收到「目前流量較高，請稍後再試」

**解決方式：** 等待一段時間後再試

### 4.3 對話歷史

- Bot 會記錄最近 **15 輪對話**
- 若長時間（超過 30 分鐘）未互動，對話會自動重置
- 重置後 Bot 不會記得之前的對話內容

### 4.4 檔案下載期限

- 整理後的檔案下載連結有效期為 **24 小時**
- 連結失效後需要重新上傳檔案處理

### 4.5 不支援的內容

| 類型 | 說明 |
|------|------|
| 群組中的圖片/檔案 | 僅一對一聊天支援 |
| 掃描型 PDF | 僅支援文字型 PDF |
| 過大的檔案 | 可能無法處理 |
| 非文字檔案 | 如圖片壓縮檔、執行檔等 |

## 5. 常見問題

### Q1: 為什麼 Bot 沒有回覆？

**可能原因：**
1. 您在群組中但沒有 @提及 Bot
2. 訊息發送過快，觸發頻率限制
3. AI 服務忙碌中
4. 網路連線問題

**解決方式：**
- 群組：確認有正確 @提及 Bot
- 等待幾秒後重試
- 查看是否有「稍後再試」的提示訊息

---

### Q2: 出現「稍後再試」的訊息怎麼辦？

這表示：
- 訊息發送過於頻繁
- AI 服務暫時忙碌

**解決方式：** 等待 **10-30 秒** 後再試

---

### Q3: 檔案下載連結失效了？

下載連結有效期為 **24 小時**。

**解決方式：**
1. 重新上傳原始檔案
2. 重新請求處理
3. 在連結失效前完成下載

---

### Q4: PDF 無法處理？

**可能原因：**
- PDF 是掃描圖片型（沒有文字層）
- PDF 已加密或損壞

**解決方式：**
- 確認 PDF 可以選取文字
- 轉換為文字檔（.txt）後重新上傳

---

### Q5: 在群組中無法使用怎麼辦？

群組使用有以下限制：
- 只回覆被 @提及 的訊息
- 不處理圖片和檔案

**解決方式：** 改為一對一聊天使用完整功能

---

### Q6: Bot 回答錯誤怎麼辦？

**說明：** AI 回覆僅供參考，可能存在不精確之處。

**建議：**
- 重要資訊請多方查證
- 可重新提問或换個方式描述問題
- 特定專業領域問題建議諮詢專業人士

---

### Q7: 如何聯繫管理員？

如有系統問題或建議，請聯繫服務管理者。
